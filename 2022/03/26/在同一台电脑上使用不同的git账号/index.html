Posted on 2022-03-26 In Git Disqus:

生成 SSH 密钥

首先需要为不同的账号生成不同的 SSH 密钥，进入 SSH 根目录下，一般是在 C:\Users\Username\.ssh ，也可以使用命令 cd ~/.ssh/ 。

使用命令创建新密钥：

ssh-keygen -t rsa -C "youremail@xxx.com"

复制 SSH 公钥到 GitHub 账户

打开 Github 上的仓库，点击 Settings > Deploy keys > Add deploy key，分别将 SSH 公钥（即 yourkeyname.pub 文件中的内容）复制到不同的 GitHub 账号仓库。

视情况勾选 Allow write access，因为我部署到第二个仓库的时候出现了 The key you are authenticating with has been marked as read only 错误，勾选之后就正常了。第一个没有勾选但是却没有出现这个报错，不知道是什么原因。
