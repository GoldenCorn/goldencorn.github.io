Scikit-learn 数据集 API 简介

sklearn.datasets

  • 加载获取流行数据集

  • datasets.load_*()：获取小规模数据集，数据包含在 datasets 里。

  • datasets.fetch_*(data_home=None)：获取大规模数据集，需要从网络上下载，函数的第一个参数表示数据集下载的目录，默认是 ~/scikit_learn_data/。

sklearn 小数据集

  • sklearn.datasets.load_iris()

    加载并返回鸢尾花数据集。

  • sklearn.datasets.load_boston()

    加载并返回波士顿房价数据集。

sklearn 大数据集

  • sklearn.datasets.fetch_20newsgroups(data_home=None,subset='train')
    • subset：’train’或者’test’，’all’，选择要加载的数据集；

数据集返回值介绍

load 和 fetch 返回的数据类型都是 datasets.base.Bunch（字典格式）。

数据集的划分

机器学习的数据集一般会划分为两个部分：

  • 训练数据：用于训练，构建模型；

  • 测试数据：在模型检验时使用，用于评估模型是否有效。
    测试集数据的比例一般占 20% ~ 30%。

数据集划分 API

  • sklearn.model_selection.train_test_split(*arrays, test_size=None, train_size=None, random_state=None, shuffle=True, stratify=None)

特征抽取

特征提取就是将任意数据（如文本或图像）转换为可用于机器学习的数字特征。

特征值化是为了让计算机更好地理解数据。

  • 字典特征提取（特征离散化）；

  • 文本特征提取；

  • 图像特征提取（深度学习）。

特征提取 API：

sklearn.feature_extraction

字典特征提取

作用：对字典数据进行特征值化，并且将属于类别的特征转换成 one-hot 编码。

  • sklearn.feature_extraction.DictVectorizer(sparse=True,...)

    • DictVectorizer.fit_transform(X)：X 代表字典或包含字典的迭代器，返回值为一个 sparse 矩阵。

    • DictVectorizer.inverse_transform(X)：X 代表 array 数组或 sparse 矩阵，返回值是转换之前的数据格式。

    • DictVectorizer.get_feature_names()：返回类别名称。

    稀疏（sparse）矩阵将非零值按其在矩阵中相应的位置表示出来，可以节省内存，从而提高加载效率。如果希望返回值是一个二维数组而非稀疏矩阵，只需要使 DictVectorizer() 方法中的参数 sparse=False 即可。

  • 应用场景：

    1. 当数据集中类别特征比较多时，就可以先将特征转换为字典类型，然后使用 DictVectorizer 进行特征提取。

    2. 本身拿到的数据就是字典类型。

文本特征提取

作用：以单词作为特征，对文本数据进行特征值化。

方法一：词频统计

  • sklearn.feature_extraction.text.CountVectorizer(stop_words=[])

    • 返回一个词频矩阵，统计每个样本特征词出现的个数。参数 stop_words 指停用词，可以数组的形式传入对分类没有什么用处的单词。

    • CountVectorizer.fit_transform(X)：X 代表文本或包含文本字符串的可迭代对象，返回值为一个 sparse 矩阵。

    • CountVectorizer.inverse_transform(X)：X 代表 array 数组或 sparse 矩阵，返回值是转换之前的数据格式。

    • CountVectorizer.get_feature_names()：返回类别名称。

注意：当文本为中文时，如果不将各个单词用空格分开，就会以短语的形式被识别统计。可使用 jieba 分词库对中文文本进行分词。

方法二：TF-IDF

由于用上述方式提取出来的单词可能对于文本分类来说并没有帮助，为了找到关键词，也就是在某类别的文章中出现频次较多，而在其他类别文章中出现频次较少的单词，可以使用另一种方法：TF-IDF。

TF-IDF 的主要思想是：如果某个词或短语在一篇文章中出现的概率高，而在其他文章中很少出现，则认为此词或短语具有很好的类别区分能力，适合用来分类。

词频（Term Frequency）指的是某一个给定的词语在该文章中出现的频率。

逆向文档频率（Inverse Document Frequency）是一个词语普遍重要性的度量。某一特定词语的 IDF 可以由 总文件数目 除以 包含该词语的文件的数目，再将得到的商取以 10 为底的对数得到。

TFIDF = TF * IDF

最终得出结果可理解为重要程度。

  • sklearn.feature_extraction.text.TfidfVectorizer(stop_words=[])

    • 返回词的权重矩阵。

    • TfidfVectorizer.fit_transform(X)：X 代表文本或包含文本字符串的可迭代对象，返回值为一个 sparse 矩阵。

    • TfidfVectorizer.inverse_transform(X)：X 代表 array 数组或 sparse 矩阵，返回值是转换之前的数据格式。

    • TfidfVectorizer.get_feature_names()：返回类别名称。

特征预处理

特征预处理就是通过一些转换函数将特征数据转换成更加适合算法模型的特征数据的过程。

当特征的单位和大小相差较大，或者某特征的方差相比其他的特征大出几个数量级时，容易影响/支配目标结果，使得一些算法无法学习到其他特征。

因此这个过程包含数值型数据的无量纲化，使不同规格的数据转换到同一规格，其中又包含两种方法：归一化、标准化。

归一化

定义：通过对原始数据进行变换把数据映射到（默认为[0,1]）之间。

该公式作用于每一列， max 为一列的最大值， min 为一列的最小值。 X'' 为最终结果， mx 和 mi 分别为指定区间值， mx 默认为 1 ， mi 默认为 0 。

  • sklearn.preprocessing.MinMaxScaler(feature_range=(0,1),...)

    • 每个特征缩放到给定范围（默认[0,1]）。

    • MinMaxScalar.fit_transform(X)：X 代表 numpy array 格式的数据[n_samples,n_features]，返回值是转换后的形状相同的 array 。

缺点：由于最大值与最小值非常容易受异常点影响，所以归一化方法鲁棒性（即稳定性）较差，只适合传统精确小数据场景。

标准化

定义：通过对原始数据进行变换把数据变换到均值为 0 ，标准差为 1 的范围内。

该公式作用于每一列， mean 为平均值， σ 为标准差，用来考量数据的稳定性。

与归一化不同的是，使用标准化方法如果出现异常点，由于具有一定数据量，少量异常点对于平均值的影响并不大，从而方差改变较小。

  • sklearn.preprocessing.StandardScaler()

    • 处理后，对每列来说，所有数据都聚集在均值为 0 附近，标准差为 1 附近.

    • StandardScaler.fit_transform(X)：X 代表 numpy array 格式的数据[n_samples,n_features]，返回值是转换后的形状相同的 array 。

标准化方法在已有样本足够多的情况下比较稳定，适合现代嘈杂大数据场景。

特征降维

降维

降维是指在某些限定条件下，降低随机变量（特征）的个数，得到一组“不相关”主变量的过程。例如将一组三维空间中的随机变量降维成在二维平面上分布的随机变量。

正因为在训练时，都是使用特征进行学习，如果特征本身存在问题或特征之间相关性较强，对于算法学习预测会影响较大。

降维方法一般分为两种：特征选择和主成分分析。

特征选择

定义：数据中包含冗余或相关变量（或称特征、属性、指标等），旨在从原有特征中找出主要特征。

Filter 过滤式

主要探究特征本身特点，特征与特征和目标值之间的关联。

可使用方差选择法来进行低方差特征过滤，或使用相关系数来衡量特征与特征间的相关程度，从而做进一步过滤处理。

  • 低方差特征过滤

    如果特征方差小，代表某个特征大多样本的值比较相近，那么需要被过滤；如果特征方差大，代表某个特征大多样本的值都有差别，需要被保留。

    • sklearn.feature_selection.VarianceThreshold(threshold = 0.0)

    • 删除所有低方差特征。

    • VarianceThreshold.fit_transform(X)：X 代表 numpy array 格式的数据[n_samples,n_features]，训练集差异低于 threshold 的特征将被删除。默认值是保留所有非零方差特征，即删除所有样本中具有相同值的特征。

  • 相关系数

    皮尔逊相关系数（Pearson Correlation Coefficient）反映变量之间相关关系密切程度的统计指标。

    相关系数的值位于 -1 与 +1 之间，即 -1 <= r <= 1 。

    • 当 r > 0 时，表示两变量正相关， r < 0 时，两变量负相关。

    • 当 |r| = 1 时，表示两变量为完全相关，当 r = 0 时，表示两变量无相关关系。

    • 一般可按三级划分： r < 0.4 为低度相关； 0.4 <= r < 0.7 为显著性相关， 0.7 <= r < 1 为高度线性相关。

    API：scipy.stats.pearsonr

Embedded 嵌入式

算法自动选择特征（特征与目标值之间的关联）。

常用方法有：决策树、正则化、深度学习等。

主成分分析（PCA）

定义：将高维数据转换为低维数据的过程，在此过程中可能舍弃原有数据、创造新的变量。

作用：数据维数压缩，尽可能降低原数据的维数（复杂度），损失尽可能少的信息。

应用：回归分析和聚类分析。

此过程的计算方式就是找到一条合适的直线，通过一个矩阵运算得出主成分分析的结果。

  • sklearn.decomposition.PCA(n_components=None)

    将数据分解为较低维数空间。

    • n_components:

      • 小数：保留百分之几的信息。

      • 整数：减少到多少特征。

    • PCA.fit_transform(X)：X 代表 numpy array 格式的数据[n_samples,n_features]，返回值是转换后指定维度的 array 。
